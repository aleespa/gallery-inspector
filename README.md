# Gallery Inspector

A powerful command-line interface (CLI) to analyze, filter, and organize your photo and video collections based on metadata (EXIF/video info).

## Features

- **Analyze**: Scan directories or individual files, extract detailed metadata (EXIF/video metadata), generate exposure/date/camera plots, and export results into a comprehensive Excel workbook.
- **Filter & Organize**: Organize files into structured directories (e.g. by Year, Month, Camera Model, or Lens) based on complex filtering queries.
- **Interactive Dashboard**: Visualize your collection's metadata with a Streamlit-based dashboard using the exported Excel workbook.
- **High Performance**: Parallel processing for fast analysis of large folders.
- **Portability**: Can be compiled into a standalone `.exe` for Windows.

## Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
   *(Note: Ensure you have `pandas`, `pillow`, `pymediainfo`, `rawpy`, `xlsxwriter`, `matplotlib`, `tqdm`, and `loguru` installed)*

## Usage

### 1. Analyze Command

Scan files/directories, export metadata to Excel, and generate visualization plots.

```bash
python main.py analyze <input_path1> [input_path2 ...] -o <output_directory> [filters]
```

**Required Arguments:**
- `inputs` (1 or more positional arguments): The directories or individual file paths to scan.
- `-o`, `--output`: The output directory where the analysis reports and plots will be saved.

**Example:**
```bash
python main.py analyze C:\Users\User\Pictures -o C:\Users\User\GalleryReports
```
This generates a folder named `Analysis YYYY-MM-DD_HH-MM-SS` in the output directory containing:
- `Metadata.xlsx`: Complete details of all analyzed images, videos, and other media files.
- `Figures/`: Data visualization plots (e.g. Exposure, Cameras, Date Distributions).

---

### 2. Filter Command

Organize and copy files from input directories into a structured hierarchy based on filters and output options.

```bash
python main.py filter <input_path1> [input_path2 ...] -o <output_directory> [filters] [organization_options]
```

**Required Arguments:**
- `inputs` (1 or more positional arguments): The directories or individual file paths to organize.
- `-o`, `--output`: The destination directory where the organized folder structure will be created.

**Organization Options:**
- `--by-media-type` / `--no-by-media-type`: Separate files into `Photos` and `Videos` top-level folders (default: `--by-media-type`).
- `--structure`: List of fields in order defining the nested folder hierarchy. Available fields: `Year`, `Month`, `Model`, `Lens` (default: `Year Month`).
- `--on-exist`: Handling of file name conflicts in destination. Choices: `rename`, `skip` (default: `rename`).
- `--verbose` / `--no-verbose`: Enable or disable verbose logging during the organization process.

**Example:**
```bash
python main.py filter C:\Users\User\Pictures -o C:\Users\User\OrganizedPhotos --structure Year Month Model --on-exist rename
```

---

### Shared Filtering Options (available on both `analyze` and `filter`)

You can apply filters to restrict which files are analyzed or organized:

| Filter Options | Description | Example |
|---|---|---|
| `--file-types` | Filter by file types (choices: `image`, `video`, `other`). | `--file-types image video` |
| `--extensions` | Filter by file extensions (case-insensitive). | `--extensions .jpg .cr2` |
| `--date-start` | Filter files taken/modified on or after this date (YYYY-MM-DD). | `--date-start 2025-01-01` |
| `--date-end` | Filter files taken/modified on or before this date (YYYY-MM-DD). | `--date-end 2025-12-31` |
| `--cameras` | Filter by camera model names (case-sensitive). | `--cameras "Canon EOS R8"` |
| `--lenses` | Filter by lens model names (case-sensitive). | `--lenses "RF24-105mm"` |
| `--min-aperture` | Filter by minimum aperture (f-number). | `--min-aperture 2.8` |
| `--max-aperture` | Filter by maximum aperture (f-number). | `--max-aperture 8.0` |
| `--min-iso` | Filter by minimum ISO value. | `--min-iso 100` |
| `--max-iso` | Filter by maximum ISO value. | `--max-iso 1600` |
| `--min-shutter-speed` | Filter by minimum shutter speed string. | `--min-shutter-speed 1/100` |
| `--max-shutter-speed` | Filter by maximum shutter speed string. | `--max-shutter-speed 1/2000` |

### Dashboard

To launch the interactive Streamlit dashboard for visualizing results:

```bash
streamlit run dashboard/app.py
```


1. **Load Data**: Load the `Metadata.xlsx` file generated by the `analyze` command.
2. **Explore**: General statistics, Sunburst charts, File Type Distribution, Exposure Triangle (ISO vs Shutter Speed), and more.
